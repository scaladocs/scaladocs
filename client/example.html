scala.util.Either[+A, +B]

2.13.x Standard Library View Source

Represents a value of one of two possible types (a disjoint union).
An instance of `Either` is an instance of either scala.util.Left or scala.util.Right.

Using Either for Flow Control:

Use Either[String, Int] to indicate whether a received input is a String or an Int. 


import scala.util.Either
import scala.util.Random

val in = if (new Random().nextBoolean()) "100" else "Some String"

val result: Either[String,Int] =
  try {
    Right(in.toInt)
  } catch {
    case e: NumberFormatException => Left(in)
  }

result match {
  case Right(x) => 
    s"You passed me the Int: $x, which I will increment. $x + 1 = ${x+1}"

  case Left(x)  => 
    s"You passed me the String: $x"
}

println(result)
